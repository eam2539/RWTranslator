# RWTranslator

RWTranslator 是一个专为 RustedWarfare 游戏 mods 设计的翻译工具。它可以帮助 mod 开发者轻松地将 mods 中的文本内容翻译成多种语言，支持多种翻译引擎，包括传统的机器翻译和先进的 LLM 翻译。

## 主要功能

### 📁 项目管理
- **导入 mods**: 支持从 ZIP 文件或文件夹导入 RustedWarfare mods 项目
- **项目列表**: 显示所有已导入的项目，支持删除和缓存清理
- **自动解压**: 导入 ZIP 文件时自动解压到项目目录

### 🔍 文件管理
- **INI 文件浏览**: 以列表形式展示项目中的所有 INI 配置文件
- **文件过滤**: 支持按文件名过滤显示的文件
- **修改状态跟踪**: 标记已修改和已查看的文件

### 🌐 翻译功能
- **多种翻译引擎**: 支持腾讯、有道、百度、Google、Microsoft、DeepL、Yandex 等翻译服务
- **LLM 翻译**: 集成 OpenAI 等大语言模型进行高质量翻译
- **批量翻译**: 支持批量翻译多个文件，提高效率
- **语言选择**: 支持多种目标语言，包括中文、英文、日文等
- **缓存机制**: 翻译结果缓存，避免重复翻译相同内容

### ✏️ 编辑功能
- **实时编辑**: 在应用内直接编辑翻译后的文本
- **段落管理**: 支持按 INI 文件的段落进行编辑
- **撤销/重做**: 支持编辑操作的撤销和重做

### 🎯 多选操作
- **批量选择**: 支持全选、反选和手动多选文件
- **批量翻译**: 对选中的多个文件执行翻译操作
- **批量导出**: 批量导出选中的翻译结果

### 📤 导出功能
- **完整导出**: 将翻译后的整个项目导出为 ZIP 文件
- **增量导出**: 只导出修改过的文件
- **自定义路径**: 支持选择导出目录

### ⚙️ 设置和配置
- **翻译引擎配置**: 配置各翻译引擎的 API 密钥和参数
- **语言偏好**: 设置默认源语言和目标语言

### 🛠️ 高级功能
- **错误处理**: 详细的错误报告和日志记录
- **性能优化**: 支持大项目的流畅操作
- **模板支持**: 自动处理 mods 中的模板占位符

## 使用方法

1. **导入项目**: 点击主界面右下角的 "+" 按钮，选择 ZIP 文件或文件夹导入 mods 项目
2. **选择文件**: 在项目列表中点击进入项目，浏览 INI 文件列表
3. **配置翻译**: 点击翻译按钮，选择翻译引擎和目标语言
4. **执行翻译**: 选择要翻译的文件，点击翻译按钮开始翻译过程
5. **编辑内容**: 点击文件进入编辑器，修改翻译结果
6. **导出项目**: 翻译完成后，点击导出按钮保存翻译后的 mods

## 系统要求

- Android 5.0 或更高版本
- 网络连接（用于在线翻译服务）
- 存储权限（用于导入和导出文件）

## 注意事项

- 部分翻译引擎需要配置 API 密钥
- 大型项目翻译可能需要较长时间
- 建议在翻译前备份原始 mods 文件
- LLM 翻译质量更高但可能产生额外费用
